About the Capstone Project

Books Checker is a web-based capstone project developed by the group Fantastic Nine from SPCC Caloocan City. It is designed to help users check and monitor the availability, price, and stock status of books in a simple and user-friendly way.

Unlike a library system, Books Checker focuses on real-time book information ‚Äî showing whether a book is available, how much it costs, and if it's still in stock or sold out. This tool aims to assist both sellers and students in making quick, informed decisions about books.

Built with modern web technologies like HTML, CSS, and JavaScript, this project reflects our team's goal to create smart and practical systems tailored for everyday use.
